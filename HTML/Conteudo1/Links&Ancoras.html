Links Externo

Usamos a tag <A> como ancora para links.

Entre no site pudim

Quando vc clica no link, ele acaba te direcionando para otro site e substitui pela sua aba, caso nao queira que isso acontessa utilizamos diversos targets.

*Caso nao tenha o link ainda para, coloque uma # para mostrar que ira preencher futuramente.

  • _self: Abre o link na mesma janela ou quadro em que o link foi clicado.
  • _blank: Abre o link em uma nova janela ou guia do navegador.
  • _parent: Abre o link no quadro pai do quadro em que o link foi clicado.
  • _top: Abre o link no quadro superior, substituindo todos os quadros existentes.

Utilize tambem rel=external para mostrar que o link vai para fora do site.
