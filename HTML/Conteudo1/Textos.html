Hierarquia de Cabecalho

A hierarquia de cabeçalhos em HTML é uma maneira de organizar o conteúdo da página em títulos e subtítulos.

Essa hierarquia ajuda a estruturar o conteúdo de forma hierárquica, onde o <h1> é o título principal e os <h2> a <h6> são os subtítulos em ordem decrescente de importância.

Por exemplo, o <h1> é normalmente usado para o título principal da página, transmitindo a ideia central ou o propósito do conteúdo.

Os <h2> são usados para subtítulos que são menos importantes que o <h1>, mas mais importantes do que os subtítulos subsequentes. O mesmo vale para os demais cabeçalhos, como <h3>, <h4>, <h5> e <h6>, cada um representando um nível de importância decrescente.

Essa hierarquia de cabeçalhos é fundamental para a estrutura e acessibilidade do conteúdo da página. Ela permite que os mecanismos de busca compreendam a organização do conteúdo e que os leitores de tela identifiquem os títulos e subtítulos para facilitar a navegação dos usuários.

Portanto, ao usar a hierarquia de cabeçalhos em HTML, é importante seguir a ordem correta, começando pelo <h1> e descendo até o <h6>. Isso garante uma estrutura lógica e coerente para o conteúdo da página, tornando-o mais fácil de ler e entender.

O que são Parágrafos?

Identacao de Linhas

A indentação de linhas em HTML refere-se ao recuo visual das linhas de código para melhorar a legibilidade e organização do código HTML. Embora a indentação não afete o funcionamento do código, ela desempenha um papel importante na clareza e manutenção do código HTML.

A indentação consiste em adicionar espaços ou tabulações no início das linhas de código HTML para criar uma estrutura visual hierárquica e facilitar a compreensão da estrutura do documento.

Listas

Lista Ordenada

Lista Ordenada <OL> é uma lista onde a ordem dos itens é algo muito importante.

*Para criar uma lista utilizamos a tag <LI>dentro da tag OL. Segue um Exemplo.

  1. Acordei
  2. Levantei da cama
  3. Escovei os dentes
  4. Tomei Cafe

A tag OL possui um párametro type, onde configuramos o tipo de marcador da lista atual. As opcoes variam de valores para esse parametro sao:

  1. Valores numerados "1"
  2. Cria Lista Por Ordem Alfabetica Maiuscula"A"
  3. Cria lista por ordem alfabetica minucsula "a"
  4. Cria Algarismo Romanos Maiusculo "I"
  5. Cria Algarismo Romanos minusculo "i"

Lista nao ordenada <UL> é uma lista onde a ordem dos itens naoé algo relevante

*Para criar uma lista utilizamos a tag <LI>dentro da tag UL. Segue um Exemplo.

  • Feijao
  • Batata
  • Carne

A tag OL possui um párametro type, onde configuramos o tipo de marcador da lista atual. As opcoes variam de valores para esse parametro sao:

  • Cria um disco padrao "DISC"
  • Cria um circulo sem estar pintado no centro
  • Cria um quadrado totalmente preto "circle"
