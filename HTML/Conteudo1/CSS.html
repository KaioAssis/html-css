HTML: Utilizando o CSS

O que é css?

O CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada em conjunto com o HTML para definir a aparência e o layout dos elementos em uma página da web. Ele permite separar o conteúdo (HTML) da apresentação (estilos) de uma forma eficiente. Com o CSS, você pode definir propriedades como cor, fonte, tamanho, margem, posicionamento e muito mais para os elementos HTML. Isso permite controlar o design e a formatação da página de maneira consistente e flexível. O CSS é aplicado aos elementos HTML usando seletores, que identificam quais elementos devem receber as regras de estilo especificadas. Além disso, o CSS suporta o conceito de cascata, o que significa que você pode definir estilos em várias camadas e os estilos mais específicos terão precedência sobre os estilos mais genéricos. Em resumo, o CSS desempenha um papel crucial na criação de páginas da web visualmente atraentes e bem estruturadas.

Formar de utilizar css

Existem tres formar de se utilizar o css dentro de um codigo em html.

  • Inline
  • Interno
  • Externo

Cada um tem sua particuliaridade, mas no fim das contas o que mais utilizamos é o externo, pois ele acaba deixando o codigo mais organizado,limpo e outras paginas podendo usar ele.

Se voce abrir todos as outras paginas apertando CTRL+U para ver o codigo da pagina vera que todas elas tao linkadas ao style.css ele que cria um estilo para todas aquelas paginas.

Abra o cosdigo da pagina para entender melhor.
