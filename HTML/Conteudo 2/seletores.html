Ao acriar uma pagina em html, podemos identificar um elemento unico com um ID, ou agruypar elementos multplicos que tenham caracteristicas semelhantes em uma Class.

Ao olhar na linha 23, vimos que o h1 se diferencia dos demais que estao na linha 24 e 27, pois ele possui um id.

Agora perceba os dois h2 que estao nas linhas 25 e 28 eles possuem o mesmo class

Em resumo, a classe é utilizada para aplicar estilos em vários elementos que possuem características em comum, enquanto o ID é usado para estilizar um único elemento de forma específica.

Toda class é identificada com ponto.
Todo id é identificado com hashtag.

Pseudo Classe e pseudo elemento.

Pseudo Classe

uma peseudo classe css é u a palavrra chave adicionada as declaracoes de um seletor apos um sinal de dois pontos e especificam um estado especial de um elemento.

alguns elementos sao:

  1. :hover;
  2. :visited;
  3. :active;
  4. :checked;
  5. :empty;
  6. :focus;

Pseudo elemento

Um pseudo elemento em CSS é uma palavra chave adicionada as declaracoes de um seletor apos dois sinais e dois pontos, e permitem que voce formate um pedaco especifico do elementro referenciado, os principais utilizados sao

  1. :before;
  2. :after;
  3. :fisrst-letter;
  4. :first-line;

Span

é usado para agrupar e estilizar um trecho específico de texto dentro de um documento HTML, sem afetar a estrutura do documento. Ele não possui significado semântico próprio, sendo usado principalmente para aplicar estilos ou identificar partes específicas do texto.
